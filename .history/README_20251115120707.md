# GitHub User Activity CLI

A **command-line tool** written in Go to fetch a GitHub user's activity including **commits, opened issues, and starred repositories**.  

This project is inspired by the [GitHub User Activity Project on roadmap.sh](https://roadmap.sh/projects/github-user-activity).
